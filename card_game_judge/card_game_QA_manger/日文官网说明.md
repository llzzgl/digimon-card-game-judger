# 日文官网QA爬取说明

## 网站结构分析

日文官网 https://digimoncard.com/rule/#qaResult_card 是一个**搜索页面**，不是直接显示所有QA的列表页。

### 页面特点

1. **需要搜索**: QA内容需要通过以下方式加载：
   - 输入卡片编号或卡片名称搜索
   - 选择分类（基本规则、关键字效果等）
   - 点击更新日期

2. **动态加载**: 内容通过JavaScript动态加载，不在初始HTML中

3. **两个QA区域**:
   - `#qaResult_card` - 卡片相关QA（需要搜索卡片）
   - `#qaResult_rule` - 规则相关QA（需要选择分类）

## 建议方案

### 方案1: 爬取规则QA（推荐）

修改URL为规则QA部分：
```python
self.base_url = "https://digimoncard.com/rule/#qaResult_rule"
```

然后选择不同的分类来爬取：
- 基本ルール（基本规则）
- キーワード効果（关键字效果）
- 用語集（术语集）
- 等等...

### 方案2: 使用中文版数据

中文版 https://app.digicamoe.cn/faq 已经成功爬取了186条QA，这些数据已经足够用于训练。

### 方案3: 手动下载官方PDF

日文官网提供PDF格式的综合规则和QA文档，可以：
1. 下载PDF文件
2. 使用PDF解析工具提取文本
3. 解析QA内容
