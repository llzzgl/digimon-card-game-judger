# 使用指南

## 当前状态

✅ 已完成中文QA爬取：**186条**

## 快速开始

### 1. 爬取日文官网QA

```bash
python scraper_jp_official.py
```

或双击运行：
```
run_jp_scraper.bat
```

### 2. 合并中日文数据

```bash
python merge_qa.py
```

### 3. 复制到微调目录

```bash
python copy_to_finetune.py
```

