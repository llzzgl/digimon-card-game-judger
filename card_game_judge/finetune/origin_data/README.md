# 源数据文件夹 (Origin Data)

本文件夹存放微调训练所需的所有源数据文件。

## 📁 文件说明

### 1. rulebook.txt
**描述：** DTCG 官方综合规则书（中文翻译版）

**来源：** 数码宝贝卡牌对战官方规则书

**用途：** 
- 提取规则解释问答
- 生成场景分析问答
- 提取关键词效果定义
- 提取效果时机说明

**格式：** 纯文本文件，包含规则编号和内容

**示例：**
```
1-3-1. 当卡牌上记载的文本内容与综合规则内容矛盾时，以文本内容为优先。
（例：通过规则，登场的数码宝贝在该回合中无法攻击，但持有≪速攻≫的数码宝贝，可以通过效果进行攻击）
```

### 3. official_qa.json
**描述：** 官方 Q&A 数据

**来源：** DTCG 官方网站、官方裁定

**用途：**
- 生成官方裁定问答
- 补充规则书未涵盖的内容

**格式：** JSON 数组，每个元素包含一个问答对

**数据结构：**
```json
[
  {
    "question": "≪贯通≫效果在什么时候触发？",

    "card_no": "",
    "card_name": "",
    "source": "官方网站",
    "date": "2025-01-01"
  }
]
```

**说明：**
- 目前包含 2 条示例数据
- 可随时添加更多官方 Q&A

## 🔄 更新数据

### 更新规则书
1. 获取最新的官方规则书（中文翻译版）
2. 保存为 `rulebook.txt`
3. 运行 `python collect_all_data.py` 重新生成训练数据

### 更新卡牌数据
1. 运行卡牌数据爬虫获取最新数据
2. 保存为 `cards.json`
3. 运行 `python collect_all_data.py` 重新生成训练数据

### 添加官方 Q&A
1. 编辑 `official_qa.json`
2. 按照数据结构添加新的问答对
3. 运行 `python collect_all_data.py` 重新生成训练数据
